# Create an omop table from a cdm table.

Create an omop table from a cdm table.

## Usage

``` r
newOmopTable(table, version = "5.3", cast = FALSE)
```

## Arguments

- table:

  A cdm_table.

- version:

  version of the cdm.

- cast:

  Whether to cast columns to the correct type.

## Value

An omop_table object
