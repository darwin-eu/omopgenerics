# Read a table from the cdm_source and add it to to the cdm.

Read a table from the cdm_source and add it to to the cdm.

## Usage

``` r
readSourceTable(cdm, name)
```

## Arguments

- cdm:

  A cdm reference.

- name:

  Name of a table to read in the cdm_source space.

## Value

A cdm_reference with new table.
