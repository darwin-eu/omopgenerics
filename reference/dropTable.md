# Drop a table from a cdm object. **\[deprecated\]**

Drop a table from a cdm object. **\[deprecated\]**

## Usage

``` r
dropTable(cdm, name)
```

## Arguments

- cdm:

  A cdm reference.

- name:

  Name(s) of the table(s) to drop Tidyselect statements are supported.

## Value

The cdm reference.
